## 📖 專案簡介 (Introduction)
你是否買了許多生產力工具卻放在角落生灰塵？或者面對龐大的任務感到焦慮拖延，遲遲無法開始？

Magic To-Do Agent 不是一個普通的待辦事項生成器。它是一個經過精心設計的 Prompt Engineering 專案，融合了《拖延心理學》、《原子習慣》以及認知行為治療（CBT）的原則。

它的核心目標不是「增加你的工作」，而是透過對話引導，降低大腦的皮質醇（壓力荷爾蒙）濃度，將模糊的焦慮轉化為可執行的「微行動」，幫助使用者找回掌控感。

## 🚀 核心特色 (Why It's Special)
這款 Agent 的設計邏輯解決了傳統 To-Do List 失敗的三大痛點：

### 1. 心理安全感建立 (Rapport & Cortisol Reduction)
痛點： 想到要做事就焦慮、想逃避。

解法： Agent 採用具備同理心與幽默感的語氣（Persona），建議使用者使用「語音模式」互動，利用神經科學原理降低防衛心，創造無壓力的對話環境。

### 2. 認知重構 (Cognitive Reframing)
痛點： 為了沒用到的工具或沒做完的事感到內疚（Guilt）。

解法： Agent 會引導使用者將「浪費」重新定義為「資源失憶症」或「待開發的潛力」。它不指責，而是將情緒內耗轉化為客觀的「病識感」，讓使用者願意面對問題。

### 3. 極致的任務拆解 (Atomic Task Decomposition)
痛點： 任務太大（如：整理人生），大腦不知道第一步該做什麼。

解法： 將抽象目標拆解為物理級別的微行動。

### ❌ 錯誤指令：「去盤點你的訂閱服務。」

✅ Agent 指令：「打開瀏覽器 -> 開啟 Google Doc -> 標題打上『資源百寶袋』。」

原理： 降低啟動阻力，讓大腦快速獲得達成任務的多巴胺獎勵。

#### 4. 遊戲化思維 (Gamification)
不只是寫清單，而是建立使用者的「能力樹」與「資源導航圖」，將枯燥的盤點轉化為角色養成的過程。

## 🛠️ 技術與設計方法 (Methodology)
本專案展示了 Prompt Engineering 在心理學應用上的潛力：

Role-Playing (角色扮演): 設定 Agent 為一位理解人類軟弱面、說話風趣的心理教練。

Chain of Thought (思維鏈): 引導 AI 先同理情緒 -> 定義問題 -> 拆解步驟 -> 執行檢核。

Interactive Guidance (互動式引導): 避免一次性輸出長篇大論，而是分階段（Phase 1, Phase 2...）釋出指令，符合人類的工作記憶（Working Memory）限制。
